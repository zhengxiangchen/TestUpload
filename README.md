# TestUpload
简化草图微信小程序<br>
## 程序功能<br>

上传草图通过对图片的线条的解析达到简化草图辅助线的功能<br>
ps:初次开发小程序,项目主要为了了解小程序,自己动手做个东西.<br>
ps:简化草图部分的代码属于机密,无法提供.<br>
